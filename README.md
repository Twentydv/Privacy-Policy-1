# Privacy-Policy-1
Политика конфиденциальности
Дата вступления в силу: 25 мая 2026 г.
Данное приложение не собирает, не хранит и не передаёт персональные данные пользователей.
Какие данные собираются
Приложение:
не требует регистрации;
не собирает личную информацию;
не отслеживает местоположение;
не использует аналитику;
не передаёт данные третьим лицам.
Разрешения
Приложение может использовать только базовые системные функции, необходимые для корректной работы часов.
Дети
Приложение безопасно для использования детьми, так как не содержит сбора данных или пользовательского взаимодействия через интернет.
Изменения политики
При изменении функционала приложения данная политика конфиденциальности может быть обновлена.
